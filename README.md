## 📖 About

The **HDI Prediction System** is a Machine Learning web application that predicts the **Human Development Index (HDI)** based on socio-economic indicators such as:

- 🌎 Country
- ❤️ Life Expectancy
- 🎓 Mean Years of Schooling
- 💰 Gross National Income (GNI) per Capita

The project combines a **React frontend** with a **Flask backend** to provide a responsive and user-friendly interface for generating HDI predictions.

---

## ✨ Features

- 📊 Predict Human Development Index (HDI)
- ⚡ Fast and responsive React UI
- 🐍 Flask REST API backend
- 🤖 Machine Learning model integration
- 📈 Categorizes countries into:
  - Low HDI
  - Medium HDI
  - High HDI
  - Very High HDI
- 🎨 Modern responsive interface

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/HDI-PROJECT.git
```

```bash
cd HDI-PROJECT
```

---

## 2️⃣ Install Frontend Dependencies

```bash
npm install
```

---

## 3️⃣ Install Backend Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Start Flask Backend

```bash
python app.py
```

---

## 5️⃣ Start React Frontend

```bash
npm run dev
```

# 📊 HDI Classification

| HDI Score | Category |
|-----------|----------|
| 0.30 – 0.40 | Low HDI |
| 0.40 – 0.70 | Medium HDI |
| 0.70 – 0.80 | High HDI |
| 0.80 – 1.00 | Very High HDI |
